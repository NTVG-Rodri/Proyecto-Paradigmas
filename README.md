# Proyecto Paradigmas de Programación

Proyecto académico desarrollado en grupo para la materia **Paradigmas de Programación** de la carrera Ingeniería en Sistemas de Información (UTN).

## Descripción
Sistema de gestión de **películas y obras de teatro**, desarrollado en **Smalltalk**, aplicando conceptos del paradigma orientado a objetos.  
El proyecto modela entidades del dominio, sus relaciones y comportamientos, incluyendo funcionalidades como administración de funciones, salas, ventas y críticas.

## Tecnologías y herramientas
- Smalltalk (VisualWorks)
- Programación Orientada a Objetos
- Modelado de dominio
- Git
- GitHub

## Conceptos aplicados
- Encapsulamiento
- Polimorfismo
- Responsabilidades y colaboración entre objetos
- Diseño orientado al dominio
